## 4.1 Descrição do Sistema 

O sistema **Quitutes da vovó** consiste em um site de vendas de doces. O objetivo do projeto é apresentar informações básicas do produto, como descrição, cardápio e formas de contato, por meio de uma interface simples e organizada, tornando-o acessível a grande maioria da sociedade.

O sistema permite que o usuário navegue entre diferentes páginas, visualize os produtos do cardápio com imagens ilustrativas e tenha acesso às informações do restaurante. O projeto foi desenvolvido utilizando apenas tecnologias de Front-End, sem uso de banco de dados ou servidor, atendendo aos requisitos propostos para a disciplina.

## 4.3 Procedimentos de Execução Local

1. Clonar o repositório ou descompactar o arquivo `.zip`.
2. Acessar a raiz do projeto.
3. Executar o arquivo `index.html` em um navegador web.
4. O sistema não requer dependências de servidor externas para seu funcionamento básico.
